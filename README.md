# Solidity Coverage Test

[![Greenkeeper badge](https://badges.greenkeeper.io/NFhbar/solcoverTest.svg)](https://greenkeeper.io/)

## Local test

clone:
```
git clone git@github.com:NFhbar/solcoverTest.git
```
install:
```
$ npm install
```
run coverage:
```
$ npm run coverage
```
 If ganache remains:
 ```
 $ npm run stop
 ```
